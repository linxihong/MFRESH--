#MFRESH空气净化器项目（个人独立开发）
#开发技术： HTML + CSS +jquery + Ajax + PHP + MySql
#项目心得：
#1.明白了在搭建项目之前需要经过非常谨慎的考虑来对整体的架构进行规划。
#2.在掌握原生JS的情况下使用jQuery简化DOM操作，提高开发效率，不断改善网页界面，努力做出用户体验更良好的页面交互。
#3.Mysql+php+ajax的合理使用能够把页面相同部分通过循环展示出来，减少相应的html代码量，直接改数据库就可以改变页面相关数据的展示。
#4.合理布置页面相关样式带来美观程度来提高用户体验度。 
#5.深深的体会到团队协作的重要性，一个完整的项目一个人短时间很难独立完成，而一个团队协作则会完成的更加高效。
